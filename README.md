# مشروع نظام تسجيل الدخول باستخدام PHP وPDO

## وصف المشروع
مشروع متكامل لنظام تسجيل الدخول وإنشاء الحسابات باستخدام PHP وPDO مع تشفير كلمات المرور.

## المميزات
- استخدام PDO للحماية من هجمات SQL Injection
- تشفير كلمات المرور باستخدام Bcrypt
- تصميم واجهة عربية متجاوبة
- نظام إدارة جلسات آمن
- تنسيق CSS عصري

## المتطلبات
- خادم ويب (Apache, Nginx, etc.)
- PHP 7.4 أو أعلى
- MySQL 5.7 أو أعلى

## خطوات التثبيت

1. قم بنسخ المشروع إلى مجلد خادم الويب
2. إنشاء قاعدة البيانات باستخدام ملف SQL المرفق
3. قم بتعديل إعدادات الاتصال في `config/database.php`
4. افتح المشروع في المتصفح
